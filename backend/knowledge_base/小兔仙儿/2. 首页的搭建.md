# Layout静态页面搭建

我们在搭建之前，都会收到UI设计的设计稿的一个静态页面，这个页面包括：

> 1. Nav区域
> 2. Header区域
> 3. 二级路口区域
> 4. Fooler区域

## 1 静态页面搭建

1. 在Layout文件夹中新建一个components文件夹，创建这三个组件文件

2. 对这三个文件进行渲染

   。。。需要手写。。。

3. 将这三个子组件引入到index.vue中

   ```vue
   <template>
       <div class="layout-index">
           <layout-nav></layout-nav>
           <layout-header></layout-header>
           <router-view>二级路由区域</router-view>
           <layout-footer></layout-footer>
       </div>
   </template>

   <script setup lang="ts">
   import { RouterView } from 'vue-router';
   import LayoutNav from './components/LayoutNav.vue';
   import LayoutHeader from './components/LayoutHeader.vue';
   import LayoutFooter from './components/LayoutFooter.vue';
   </script>
   ```

## 2 字体图标引入

官网:[iconfont-阿里巴巴矢量图标库](https://www.iconfont.cn/)

1. 查看引入方式：在官网的帮助/代码应用（[iconfont-阿里巴巴矢量图标库](https://www.iconfont.cn/help/detail?spm=a313x.manage_type_myprojects.i3.48.33363a81ILCVGD&helptype=code)）。我们使用的是font-class 引用方法。

2. 选取需要的**所有**字体图标及生成fontclass代码：

   > 1. 在首页中直接搜索即可（如购物车、搜索、分享）；
   >
   > 2. 放入购物车，直到选完为止；
   >
   > 3. 点击导航条的购物车，添加至项目
   >
   > 4. 加入或新建一个项目
   >
   > 5. 选中资源管理->我的项目->Font class
   >
   >    ​                                          ->在线查看链接->复制

3. 引入到项目中：

   将刚刚复制的链接粘贴到`index.html`

   ```html
   <link rel="stylesheet" href="//at.alicdn.com/t/font_2143783_iq6z4ey5vu.css">
   ```

4. 使用字体图标：

   在任意组件中进行嵌套`<i class="iconfont icon-xxx"></i>`，xxx是类名（也就是我之前选取的Font class）

   ```vue
   <template>
     <dd><i class="iconfont icon-weixin"></i> 公众号</dd>
     <dd><i class="iconfont icon-weibo"></i> 微博</dd>
   </template>
   ```

> 注意：
>
> ​	这个链接是可复用的，可在不同设备正常使用

**实现步骤**

1. 封装接口函数

   `apis/Layout.js`

   ```js
   import httpInstance from '@/utils/http'

   export function getCategoryAPI () {
     return httpInstance({
       url: '/home/category/head'        //url参数在接口文档里，因为是GET请求，所以省略了get
     })
   }
   ```

2. 调用接口函数

   `LayoutHeater.vue`

   ```vue
   <script setup>
     import { getCategoryAPI } from '@/apis/layout'  //第一步：引入接口
     import { onMounted, ref } from 'vue'

     const categoryList = ref([])
     const getCategory = async () => {             //第二步：编写获取数据的函数，并放入一个空数组
       const res = await getCategoryAPI()
       categoryList.value = res.result
     }

     onMounted(() => getCategory())               //第三步：使用生命周期函数，调用获取数据
   </script>

   <template>
     ...                                          //第四步：拿到数据后进行渲染 
           <li class="home" v-for="item in categoryList" :key="item.id">
             <RouterLink to="/">{{ item.name }}</RouterLink>
           </li>
     ...
   </template>
   ```

## 5 实现吸顶导航

吸顶导航指的是上拉页面后，将导航固定在顶部

会使用到VueUse([useScroll | VueUse 中文网](https://vueuse.nodejs.cn/core/useScroll/))

1. 这个导航需要单独重新写一个，一般要比之前那个要窄一些

   另外，这个导航需要单独放在一个组件中`views/components/LayoutEixed.vue`

2. 这个导航最开始：往上平移自身高度 + 完全透明；之后：移除平移 + 完全不透明

   `views/components/LayoutEixed.vue`

   ```vue
   <style scoped lang='scss'>
   .app-header-sticky {
     ...
     // 状态一：往上平移自身高度 + 完全透明
     transform: translateY(-100%);
     opacity: 0;

     // 状态二：移除平移 + 完全不透明
     &.show {
       transition: all 0.6s linear;
       transform: none;
       opacity: 1;
     }
     ...
   }
   <style>
   ```

3. 通过添加添加类名实现切换

   `views/components/LayoutEixed.vue`

   ```vue
   <script setup>
   import { useScroll } from '@vueuse/core'
   const { y } = useScroll(window)
   </script>

   <template>
     <div class="app-header-sticky" :class="{ show: y>100}">
       ...
     </div>
   </template>
   ```

   这里需要我们获取向上滚动的距离，然后根据距离再添加类名

   1. 获取距离：

      使用VueUse([useScroll | VueUse 中文网](https://vueuse.nodejs.cn/core/useScroll/))的useScroll函数，用法查看官网即可

      通过解构赋值的方式获得距离；注意对象是`window`

   2. 大概估计一个距离，进行添加类名

## 6 使用pinia优化重复请求

因为我们上边写了两次一级导航，也就请求了两次，为避免这种重复请求，我们可以将这个数据**集中触发**，再分别拿到数据。

1. 新建category文件，将**调用接口**函数写入pinia

   `stores/category.ts`

   ```ts
   import { ref } from 'vue'
   import { defineStore } from 'pinia'
   import { getCatgoryAPI } from '@/apis/Layout'

   export const useCategoryStore = defineStore('category', () => {
     //导航列表数据管理
     //state导航列表数据
     const categoryList = ref([])
     //cation获取导航列表方法
     const getCategory = async() => {
     const res = await getCatgoryAPI() 
     categoryList.value = res.result
     }

     return { 
       categoryList,
       getCategory
     }
   })
   ```

2. 在他们共同的父组件集中调用

   `Layout/index.vue`

   ```js
   //触发获取导航列表的action
   import { useCategoryStore } from '@/stores/category'
   import { onMounted } from 'vue';
   const categoryStore = useCategoryStore()
   onMounted(() => categoryStore.getCategory())
   ```

3. 在需要的组件进行使用

   例如`Layout/components/LayoutHeater.vue`

   ```vue
   <script setup>
   import { useCategoryStore } from '@/stores/category'
   const categoryStore = useCategoryStore()          //注意：获取数据变量与调用变量相同
   </script>

   <template>
     ...
           <li class="home" v-for="item in categoryStore.categoryList" :key="item.id">
             <RouterLink to="/">{{ item.name }}</RouterLink>
           </li>
    ...     
   </template>
   ```
