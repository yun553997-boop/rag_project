# 项目起步

## 1 创建项目

1. 创建项目文件

   ```bash
   npm init vue@latest
   ```

2. 安装依赖

   ```bash
   npm install
   ```

3. 查看jsconfig.json是否配置别名

   `tsconfig.app.json`

   ```json
   {
     "compilerOptions" : {
       ...
       "paths" : {
         "@/*":["src/*"]
       }
     }
   }
   ```

## 2 引入elementPlus

网址：[快速开始 | Element Plus](https://element-plus.org/zh-CN/guide/quickstart)

1. 安装elementPlus

   ```bash
   npm i elementplus
   ```

   按照流程---按需引入

2. 自动导入插件

   首先你需要安装`unplugin-vue-components` 和 `unplugin-auto-import`这两款插件

   ```bash
   npm install -D unplugin-vue-components unplugin-auto-import
   ```

   然后把下列代码插入到你的 `Vite`配置文件中

   ```js
   // vite.config.js 示例
   import AutoImport from 'unplugin-auto-import/vite'
   import Components from 'unplugin-vue-components/vite'
   import { ElementPlusResolver } from 'unplugin-vue-components/resolvers'

   export default defineConfig({
     plugins: [
       // 自动导入 API
       AutoImport({
         resolvers: [ElementPlusResolver()],
       }),
       // 自动导入组件
       Components({
         resolvers: [ElementPlusResolver()],
       }),
     ],
   })
   ```

## 3 定制elementPlus主题颜色

网址：[主题 | Element Plus](https://element-plus.org/zh-CN/guide/theming)

1. 安装sass

   ```bash
   mpm i sass -D 
   ```

2. 在src新建文件`styles/element/index.scss`

   在文件中写入自己的主题颜色，例如小兔鲜主题颜色：

   ```scss
   /* 只需要重写你需要的即可 */
   @forward 'element-plus/theme-chalk/src/common/var.scss' with (
     $colors: (
       'primary': (
         // 主色
         'base': #27ba9b,
       ),
       'success': (
         // 成功色
         'base': #1dc779,
       ),
       'warning': (
         // 警告色
         'base': #ffb302,
       ),
       'danger': (
         // 危险色
         'base': #e26237,
       ),
       'error': (
         // 错误色
         'base': #cf4444,
       ),
     )
   )
   ```

3. 在vite配置文件中覆盖默认主题颜色

   `vite.config.ts`

   ```typescript
   export default defineConfig({
    ...
       Components({    //配置elementPlus采用sass样式配色系统
         resolvers: [ElementPlusResolver({importStyle: "sass"})],
       })
     ],
     resolve: {...},
     css: {
       preprocessorOptions: {
         scss: {
            //自动导入定制化样式进行样式覆盖
           additionalData: `@use "@/styles/element/index.scss" as *;`,
         },
       },
     },
   })
   ```

## 5 设置路由

路由设计原则：找页面的切换方式，如果是整体切换，则为一级路由，如果是在一级路由的内部进行的内容切换，则为二级路由

1. 创建路由文件：`Login`、`Layout `、`Home `、`Category`，并分别写入测试标签文字

2. 配置路由

   `router/index.ts`

   ```ts
   1.分别引入这些组件
   import Category from '@/views/Category/index.vue'
   import Home from "@/views/Home/index.vue"
   import Layout from '@/views/Layout/index.vue'
   import Login from '@/views/Login/index.vue'

   2.配置路由规则
   const router = createRouter({
     history: createWebHistory(import.meta.env.BASE_URL),
     routes: [
       ...
     ]
   })
   ```

3. 在这些路由组件的上一级中展示，例如

   `APP.vue`

   ```vue
   <template>
     //一级路由展示区
     <router-view></router-view>
   </template>
   ```

## 6 静态资源引入

静态资源分为图片资源和样式资源

1. 图片资源一般有UI设计师提供，直接放入到`assets`文件夹即可

2. 样式资源一般是项目初始化的时候进行重置，需要自己手写或开源的`normalize.css`

   这个样式文件放在`slytes`文件夹中，注意：需要在入口文件引入才能生效

## 8 scss文件自动导入

​        在项目中一些组件共享的色值会以变量的方式统一放入到一个`var.scss`的文件中，正常组件使用，需要先导入scss文件，在使用内部变量。  

​	自动导入，可以免去手动导入文件的步骤，直接使用内部变量

1. 将`var`文件放入slytes文件夹中

   `var.scss`

   ```scss
   $xtxColor: #27ba9b;
   $helpColor: #e26237;
   $sucColor: #1dc779;
   $warnColor: #ffb302;
   $priceColor: #cf4444;
   ```

2. 在`vite.config.ts`中导入

   ```ts
    css: {
       preprocessorOptions: {
         scss: {
           //自动导入定制化样式进行样式覆盖
           additionalData: `
           @use "@/styles/element/index.scss" as *;  
           @use "@/styles/var.scss" as *;
            `,   
         },
       },
     },
   ```
