# 二级路由导航

点击首页外的其他一级导航的下面内容时跳转的内容（首页>居家>宠物食品，即宠物食品的导航），但是本章只在“全部分类”添加了`</RouterLink>`

> 本来的意思是这样的：点击“全部分类”下的任意内容，实现区别性跳转，然后来到二级路由，在二级路由中又有二级导航，然后点击任意导航，实现区别性跳转
>
> 但是，由于这个项目接口没有那么多充足数据
>
> 故而，在二级路由只有一份**单一数据**（没有三个导航下的内容数据）

## 1 二级路由模板搭建

1. 准备组件模版（这一步可以先不做，因为这是模板内容，但要先**建立**这个**文件**并填充）

   `views/subCategory/index.vue`

   ```vue
   <!-- “全部分类”下的二级路由 -->
   <template>
     <div class="container ">
       <!-- 面包屑 -->
       <div class="bread-container">
         <el-breadcrumb separator=">">
           <el-breadcrumb-item :to="{ path: '/' }">首页</el-breadcrumb-item>
           <el-breadcrumb-item :to="{ path: `/category/${categoryfilterData.parentId}` }">
               {{ categoryfilterData.parentName }}
           </el-breadcrumb-item>
           <el-breadcrumb-item>{{ categoryfilterData.name}}</el-breadcrumb-item>
         </el-breadcrumb>
       </div>
       <!-- 面包屑下的展示组件 -->
       <div class="sub-container">
         <!-- 二级导航 -->
         <el-tabs v-model="reqDate.sortField" @tab-change="tobChange">
           <el-tab-pane label="最新商品" name="publishTime"></el-tab-pane>
           <el-tab-pane label="最高人气" name="orderNum"></el-tab-pane>
           <el-tab-pane label="评论最多" name="evaluateNum"></el-tab-pane>
         </el-tabs>
         <!-- 商品展示 -->
         <div class="body" v-infinite-scroll = "load" :infinite-scroll-disabled="disabled">
           <!-- 单元商品组件 -->
           <Gooditem v-for="good in goodList" :key="good.id" :good="good"></Gooditem>
         </div>
       </div>
     </div>
   </template>
   ```


1. > 注意：这里的面包屑与前面（一级导航）的面包屑是两个完全没有关系的两个面包屑，这两个看似是一个一级，一个二级，实则都是二级路由，因为只有一个二级路由出口，且同时只能展示一个

2. 配置路由关系

   路由传参先占位

   ```js
   。。。
     routes: [
       {

         component: Layout,
         children: [
           ...
           {

             component: SubCategory

         ]

   ```


3. 跳转配置

   在category（一级导航下的区域）中设置（首页>居家>  ?  ），设置这里向哪里（<RouterLink :to="`/category/sub/${i.id}`">）跳转

   `views/Category/index.vue`

   ```vue
   <div class="sub-list">
     <h3>全部分类</h3>
     <ul>
       <li v-for="i in categoryData.children" :key="i.id">
   ```

   > 因为这里我们只给全部分类设置了二级路由调转的标签（` <RouterLink />`）,所以：
   >
   > ​	前面我们设置的模板才会展示在这里；
   >
   > ​	点击这里才会跳转二级路由（当然在其他模板（首页）也有这个标签，但     `Category/index`只有这一个）

## 2 二级路由模板内容填充

### 2.1 面包屑导航实现

1. 准备接口

   `apis/category.ts`

   ```ts
   export const getCategoryFilterAPI = (id) => {
     return httpInstance({
       url:'/category/sub/filter',
       params:{
         id
       }
     })
   }
   ```


2. 获取数据渲染模版

   这里与前面基本相同（通过路由传参，并在设置请求函数这里接收参数）

   `views/subCategory/index.vue`

   ```vue
   <script setup>
   import { getCategoryFilterAPI } from '@/apis/category'
   // 获取二级导航数据
   const filterData = ref({})
   const getFilterData = async () => {
     const res = await getCategoryFilterAPI(route.params.id)
     filterData.value = res.result
   }
   getFilterData()     //当然这里应该还是用钩子函数触发
   </script>

   <template>
     ...              
         <el-breadcrumb-item :to="{ path: `/category/${filterData.parentId}` }">{{ filterData.parentName }}      
         </el-breadcrumb-item>
         <el-breadcrumb-item>{{ filterData.name }}</el-breadcrumb-item>
    ...
   </template>
   ```



### 2.2 商品填充

大致内容与前面无异

1. 设置请求函数

   `apis/category.js`

   ```js
   export const getSubCategoryAPI = (data) => {
     return httpInstance({
       url:'/category/goods/temporary',
       method:'POST',
       data
     })
   }
   ```

2. 拿到请求函数并触发

   `subCategory/index.vue`

   ```ts
   const goodList = ref([])
   //设置参数
   const reqDate = ref({

     page:1,
     pageSize:20,

   ```

3. 使用

   ```vue
       </div>
     </div>
   </template>
   ```

### 2.3 二级导航内容切换

> 思路：tab组件切换时修改reqData中的sortField字段，重新拉取接口列表
>
> 因为只有一份**单一数据**，所以没有实现跳转（内容都一样）

这里通过双向绑定内容，实现点击拿到对应字段，再通过查阅[Tabs 标签页 | Element Plus](https://element-plus.org/zh-CN/component/tabs)`activeName` 改变时触发的事件

```vue
<!-- 二级导航 -->
<el-tabs v-model="reqDate.sortField" @tab-change="tobChange">
  <el-tab-pane label="最新商品" name="publishTime"></el-tab-pane>
  <el-tab-pane label="最高人气" name="orderNum"></el-tab-pane>
  <el-tab-pane label="评论最多" name="evaluateNum"></el-tab-pane>
</el-tabs>
```

```ts
//设置二级导航下的列表数据切换
const tobChange = () => {
  reqDate.value.page = 1 //重置page(当前页面)
  getGoodList()          //因为接口没有这三项数据，所以就临时借用getGoodList() 数据
  // reqDate.value.sortField = ...   真实情况下会通过切换这个,来切换这三个导航下的内容
}
```

### 2.4 商品列表无限加载

无限添加本质：通过接口数据一页一页的**添加**数据，直到添加到请求不到数据

（我们接收的参数page: 1,pageSize: 20表示当前第几页,一页多少数据，如，page=1就是第一页的数据，page=2就是第二页数据）

> 基础思路
>
> 1. 触底条件满足之后 page++，拉取下一页数据
> 2. 新老数据做数组拼接
> 3. 判断是否已经全部加载完毕，停止监听

通过[Infinite Scroll 无限滚动 | Element Plus](https://element-plus.org/zh-CN/component/infinite-scroll)实现无限滚动+禁止加载

```ts
<script setup>
  // 加载更多
  const disabled = ref(false)
  const load = async () => {
    reqData.value.page++                              //注意这里先自加page
    const res = await getSubCategoryAPI(reqData.value)//再进行加载
    goodList.value.push(...res.result.items)          //数组添加
    // 加载完毕 停止监听
    if (res.result.items.length === 0) {
      disabled.value = true
    }
  }
</script>

<!-- 商品展示 -->
<div class="body" v-infinite-scroll = "load" :infinite-scroll-disabled="disabled">
  <!-- 单元商品组件 -->
  <Gooditem v-for="good in goodList" :key="good.id" :good="good"></Gooditem>
</div>
```

## 3 定制路由滚动行为

问题：

​	滚动二级路由内容后，再点击一级导航，跳至一级路由内容会在原先二级路由滚动条的位置。

我们想要的效果：

​	点击后跳转至顶部

如何解决：

​	vue-router内置scrollBehavior配置项，指定路由切换时的滚动位置

`router/index.ts`

```ts
routes: [
    ...
  ],
  //路由滚动行为定制
  scrollBehavior() {
    return {
      top: 0
    }
  }
```
