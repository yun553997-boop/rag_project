## 2 渲染数据

1. 封装接口

   拿到该组件的数据

   `apis/detail.js`

   ```js
   //商品详情
   export const getDetail = (id) => {
       return httpInstance({
           url:'/goods',
           params:{ id }
       })
   }
   ```

2. 获取数据渲染模版

   这里与前面获取数据方法一样,使用自定义hook

   ```js
   import { getDetail } from '@/apis/detail'
   import { onMounted, ref } from 'vue'
   import { useRoute } from 'vue-router'

   export const useGoods = () => {
       const goods = ref({})
       const route = useRoute()
       const getGoods = async () => {
         const res = await getDetail(route.params.id)
         goods.value = res.result
       }
       onMounted(() => getGoods())

       return {
           goods
       }
   }
   ```

#### 2.1.1 渲染基本模板

2. 封装接口

   `apis/detail.js`

   ```js
   //商品详情中的每日榜单
   export const getHotGoodsAPI = ({id, type, limit=3}) => {
       return httpInstance({
           url:'/goods/hot',
           params:{ 
               id,        // 商品id
               type,      // 1代表24小时热销榜 2代表周热销榜
               limit      // 获取个数(默认为3个)
            }
       })
   }
   ```


3. 获取基础数据渲染模版

#### 2.1.2 适配热榜类型

通过父传子,  传递`type`参数（1代表24小时热销榜 2代表周热销榜 3代表总热销榜）

1. 子

   `Detail/components/DetailHot.vue`

   ```ts
   const props = defineProps({
     hotType: {
       type: Number,  
       default: 1  //默认为1（24小时热销榜）
     }
   })
   const title = props.hotType===1?'24小时热榜':'周热榜'
   const res = await fetchHotGoodsAPI({
       id: route.params.id,
       type: props.type               //别忘了，在这里传递，否则就只是更换了title
   })
    
   <template>
       <h3>{{ title }}</h3> 
   </template>
   ```


2. 父

   `Detail/inex`

   ```vue
   <!-- 24小时 -->
   <DetailHot :hot-type="1"></DetailHot>   //注意：这里有冒号（否则1就是字符串了）
   <!-- 周 -->
   <DetailHot :hot-type="2"></DetailHot>
   ```
