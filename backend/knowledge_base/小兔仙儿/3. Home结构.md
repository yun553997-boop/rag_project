## 4 图片懒加载

### 4.1 封装全局指令

官网：[自定义指令 | Vue.js](https://cn.vuejs.org/guide/reusability/custom-directives.html#custom-directives) 

​           [useIntersectionObserver | VueUse 中文网](https://vueuse.nodejs.cn/core/useIntersectionObserver/)

自定义一个全局指令，使用一个钩子函数（mounted()）进行触发，怎么触发使用useIntersectionObserver方法，判断是否进入视口区域，进入视口区域触发请求，请求之后进行赋值

`main.ts`

```ts
// 定义懒加载插件
import { useIntersectionObserver } from '@vueuse/core'

//定义全局指令
app.directive('img-lazy',{
    mounted (el,binding) {
        //el:指令绑定的元素：img
        //binding:bindinf.value  绑定表达式的值：图片url
        useIntersectionObserver(   //监听是否进入视口
            el,  
            //进入视口    
            ([{ isIntersecting }]) => {
                if ( isIntersecting ) {
                    el.src = binding.value
                }
            }
        )
    }
})
```

`HomeHot.vue`

```vue
<img :src="item.picture" alt="" /> //原来的


<img v-img-lazy="item.picture" alt="">   //触发请求并赋值
```
